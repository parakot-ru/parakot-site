# Parakot Site

Монорепозиторий проекта `parakot.ru`.

## Структура

```text
landing/  - публичный лендинг
admin/    - React SPA-админка
api/      - PHP REST API
docs/     - ТЗ, архитектура и заметки по проекту
```

## Что внутри

- Лендинг читает управляемый контент из API.
- Админка управляет настройками, контактами, секциями, карточками и заявками.
- API хранит данные в MySQL и принимает заявки с формы.
- Seed-скрипты помогают быстро заполнить стартовый контент.

## Локальная работа

```bash
cd admin
npm install
npm run dev
```

Публичную часть можно смотреть напрямую:

```text
landing/index.html
```

Локальные секреты API лежат в `api/.env` и не попадают в git.
