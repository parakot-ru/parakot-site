# Parakot.ru - ТЗ на одностраничный сайт

## 1. Цель проекта

Сделать новый одностраничный сайт для Константина "Кота" / Parakot про:

- обучение полетам на параплане
- туры и выезды по Кавказу
- повышение уровня для пилотов

Задача сайта:

- создать доверие к инструктору
- красиво показать атмосферу полетов и гор
- собирать заявки в Telegram / по телефону / через форму
- заменить старый сайт на новом домене `parakot.ru`

## 2. Формат сайта

Рекомендуемый формат:

- атмосферный одностраничный лендинг
- адаптивный под мобильные и десктоп
- быстрый, легкий, без перегруженной CMS

Рекомендуемое решение:

- статический сайт на HTML/CSS/JS или на легком фронтенд-стеке
- размещение на Beget
- подключение домена `parakot.ru` после утверждения контента

Почему не Tilda:

- меньше гибкости по дизайну
- лишняя абонентская зависимость
- сложнее сделать "не шаблонно"

Почему не WordPress по умолчанию:

- для лендинга избыточен
- требует обновлений и обслуживания
- больше риск перегруза плагинами

Когда WordPress все же можно брать:

- если планируется блог
- если Константин сам будет регулярно публиковать новости, даты сборов и статьи

## 3. Позиционирование

Ключевая идея:

`Не просто школа полетов, а личный сайт опытного инструктора и гида по лучшим летным местам Кавказа.`

Тон сайта:

- спокойный
- уверенный
- живой
- без пафоса
- с акцентом на опыт, безопасность, горы и настоящее сообщество

## 4. Целевая аудитория

### Новички

Люди, которые хотят начать летать с нуля, но не понимают, как устроено обучение и кому можно доверять.

### Продолжающие пилоты

Люди после базового обучения, которые хотят расти дальше: динамик, термики, маршрутное мышление, новые места.

### Самостоятельные пилоты

Пилоты, которым интересны туры, выезды и новые локации Кавказа вместе с опытным гидом.

## 5. Основные смыслы сайта

Сайт должен доносить:

- у Константина большой реальный опыт
- он не продает "аттракцион", а дает системное обучение
- Кавказ - сильный и разнообразный регион для полетов
- формат подходит и новичкам, и опытным пилотам
- программа подбирается по погоде и уровню участника
